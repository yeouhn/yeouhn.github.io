basic workflow

git status to discover which file(s) have been changed since the last save
git diff filename to compare the file as it currently is to what you last saved
  lines being removed are prefixed with -, lines being added are prefixed with +
  @@ tells where the changes are being made. the pair of numbers are start line and number of lines
git diff will show you all the changes in your repository
git diff directory will show you the changes to the files in some directory
git add filename to add a file to the staging area
